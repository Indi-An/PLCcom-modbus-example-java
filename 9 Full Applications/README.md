# 9 Full Applications - Java

The full applications are richer manual validation tools. They are not the first place to learn the API, but they are very useful once a device, simulator or customer scenario needs to be inspected interactively.

## Applications

| # | Project | What it is for |
|---|---------|----------------|
| 91 | `91_Full_Master_Sample` | Swing master application for TCP, SecureTCP, register modes, byte order and request testing. |
| 92 | `92_Full_Slave_Sample` | Swing slave application with normal TCP and SecureTCP listeners, data store view and certificate manager. |

## When To Use Them

Use the small console workshops first when you want to understand one idea. Use the full applications when you need manual control, visual feedback, log output, certificate inspection or a convenient way to compare behavior against a real Modbus device.

The full applications still remain examples. Review credentials, write operations, TLS behavior and certificate handling before adapting ideas for production code.
